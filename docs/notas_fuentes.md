## Advertencias metodológicas

1. Las cifras citadas en los documentos no siempre tienen fuente explícita. Se debe verificar cada una contra fuentes oficiales (UNODC, DIAN, Policía Nacional, OCHA, CODHES).
2. El sitio web puede actualizarse durante el análisis. Guardar fecha de consulta para cada extracción.
3. La voz narrativa alterna entre primera persona (candidato habla directamente) y tercera persona institucional. Distinguir en el análisis.
4. Los documentos combinan propuestas propias con ataques al gobierno actual — separar ambas dimensiones en la clasificación.
